Texts:

This section describes the text formatting and their meaning.

Alerts

Alerts box are used to draw the user attention to an information.

Alert box should contain a <h1> item that summarize what it's about. And can contain one or mode <p> to provide more details.

Short description

The info class is used to accent some relevant information.

<div class"alert info">
<h1>Short description</h1>
<p>The <code>info</code> class is used to accent some relevant information.
</div>
